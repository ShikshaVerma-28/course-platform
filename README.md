# 🎓 Online Learning Platform

A comprehensive, responsive educational portal designed to provide a seamless learning experience, featuring course catalogs, student dashboards, and easy navigation.

![Project Preview](Screenshot%202026-06-14%20160500.png)

## ✨ Features
* **Dynamic Course Catalog:** Explore a wide variety of subjects with an intuitive filtering and viewing system.
* **Student Dashboard:** Personalized interface for tracking progress and accessing enrolled courses.
* **Fully Responsive:** Optimized performance and layout for learning on the go—anytime, anywhere.
* **Modern UI/UX:** Clean, distraction-free design focused on enhancing educational engagement.

## 🛠️ Tech Stack
* **Markup:** HTML5
* **Styling:** CSS3
* **Logic:** JavaScript

## 📦 Quick Start

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/ShikshaVerma-28/Course-Platform.git](https://github.com/ShikshaVerma-28/Course-Platform.git)
   ```
   
2. **Navigate to the directory:**
   ```bash
   cd Course-Platform
   ```
   
3. **Run the project:**
Open index.html in your browser to start exploring.
